## 软件介绍

- 自动化构建项目（告别重复劳动）
- 规范项目结构（只要创建，就是规范）
- 自动管理依赖构件（告别jar版本混乱）
- 一键发布项目/基线（告别各种脚本）


　快读发布一个可运行于生成环境的应用程序（继承运行环境、部署文档、版本说明、启动脚本等）

> 支持混淆打包（使用说明待补充）
